<SpecSection title="Scope" id="scope">
This **language-meta area** defines:

- The **`Meta` project type** — how a manifest declares a project that **pipes into the compilation process** for meta orchestration (see **[Project manifest contract](/platform-spec/tooling/manifests-and-lockfiles/project-manifest-contract/)** for schema).
- The **`meta` compilation block** — declarative **capture → process → emit** pipelines over the **current compilation**, with deterministic, non-destructive iteration semantics.

**Implementation** — `Beskid.Compiler.*` module shape, host execution, merge, diagnostics transport, and incremental invalidation — is specified under **[Metaprogramming Mod SDK](/platform-spec/compiler/metaprogramming-mod-sdk/)**.
</SpecSection>
