This feature hub defines **`Beskid.Compiler.Process`** and host execution: **Meta** project registration, **compilation events**, **named process contexts**, capture/process/emit scheduling, and capability policy.

## Language alignment
Implements language-meta **capture → process → emit** for `meta` items and manifest-driven **`Meta`** orchestration.

## Implementation anchors
- `compiler/crates/beskid_engine/` — JIT host patterns reusable for compile-time execution.
- `compiler/crates/beskid_runtime/` — sandboxed builtins surface for meta workloads.
- `compiler/crates/beskid_analysis/src/services.rs` — compilation session orchestration hooks.
