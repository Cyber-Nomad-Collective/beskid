1. Resolve source path/text through services.
2. Parse program and produce syntax diagnostics.
3. Run builtin staged semantic rules when diagnostics are enabled.
4. Abort on error diagnostics.
5. Transform to HIR, resolve symbols, and type-check.
6. Lower typed program to backend artifact for JIT/AOT.

The canonical implementation entrypoint is `beskid_codegen::services::lower_source`.
