---
title: "beskid tree"
description: "Print a textual AST tree for a Beskid source file."
---

Parse one `.bd` file and print a hierarchical visualization of the AST (same parse path as [`parse`](/guides/cli/commands/parse/)).

## Arguments

| Argument | Description |
| --- | --- |
| `<INPUT>` | Required path to a `.bd` file |

## Example

```bash
beskid tree src/Main.bd
```
