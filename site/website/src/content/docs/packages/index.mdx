## Package operations at a glance

This hub ships a full UI workflow for package operators:

- **Listing**: searchable package catalog with status and risk signal.
- **Reviewing**: triage queue with moderation outcomes.
- **Managing**: owner/visibility policy updates in one form.

It also documents the compiler-integrated package workflow:

- **CLI client** for auth, add/remove/install, publish, and yanking.
- **Workspace orchestration** with `Workspace.proj`.
- **Artifact format** (`.bpk`) and lockfile reproducibility model.
- **Registry protocol** contracts and compatibility policy.
